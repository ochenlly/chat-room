## chat-room 😀

### 技术栈

1.前端

- vue3
- vite
- vue-router
- socket.io-client

2.后端

- nodejs
- express
- socket.io

### 实现功能 ✔

- 发送消息带有时间和发送者昵称
- 实现组播，发送的消息同步给正处于聊天室的用户
- 登录与退出聊天室

### 不足

- 私发消息没有实现
- 在线的人数没有统计，没有展示在线的所有人信息
- 页面没有认真写，单纯就是为了实现聊天室的功能 😜
